== Description ==

Popularis Extra gives you access to demo import for free WordPress theme - [Popularis](https://wordpress.org/themes/popularis/), extra features like widgets, shortcodes or additional Elementor widgets.

This plugin requires the [Popularis](https://wordpress.org/themes/popularis/) theme to be installed.

== Installation ==

1. Upload `popularis-extra` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
